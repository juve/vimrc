Gideon's Vim Configuration
==========================

To use:

    $ cd
    $ git clone http://github.com/juve/vim.git .vim
    $ echo "source ~/.vim/vimrc" > .vimrc

You also have to compile the command-t extension:

    $ cd ~/.vim/bundle/command-t/ruby/command-t
    $ ruby extconf.rb
    $ make
